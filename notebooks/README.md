# Notebooks

Use esta pasta para exploração, validação e análises pontuais.

Recomendações:

1. Não colocar lógica produtiva somente em notebooks.
2. Migrar transformações reutilizáveis para `src/`.
3. Nomear notebooks com prefixos:
   - `00_` sandbox
   - `01_` EDA
   - `02_` validação
   - `03_` storytelling
